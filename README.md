# SaveFrame

## インストール方法
`.jsx` ファイルを **ScriptUI Panels** フォルダに入れ、After Effects の「ウィンドウ」メニューから実行してください。

## 機能
現在表示しているフレームのスクリーンショットを直接保存できます。
